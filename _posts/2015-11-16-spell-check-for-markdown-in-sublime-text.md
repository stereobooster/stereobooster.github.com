title: Spell Check for Markdown in Sublime Text
tag:
 - tips
---

Open any markdown file. Go to: `Preferences` → `Settings - More` → `Syntax Specific - User`.

This will open up a file named: `Markdown.sublime-settings`.

Add the following:

```json
{
  "spell_check": true
}
```

Save file.
